Project Inspiration

I was working on a project once where I was asked to extract text information from 100 Images, well the most illogical way to do that would have been to type them out one after the other. But thanks to my knowledge of python and data extraction, I was able to codify the project and finish it within 30mins. Then the thought of turning it into a web app for ease of access to anyone. I hope you find this project worth using for your simple text extraction from images.
